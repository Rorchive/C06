## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/Rorchive/C06.git
```

2. Navegue até o exercício desejado:
```bash
cd "C06/01 - Tipos Primitivos e Controle de Fluxo/01 - Zé do Lanche"
```

3. Compile e execute o arquivo Java:
```bash
javac Main.java
java Main
```

> **Nota:** Certifique-se de ter o [JDK](https://www.oracle.com/java/technologies/downloads/) instalado e configurado no seu PATH.
